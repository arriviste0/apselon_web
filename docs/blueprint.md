# **App Name**: Apselon Technologies

## Core Features:

- Hero Section with Animation: Engaging hero section with company info, values, CTAs (Request Quote, Download Capabilities Sheet), and image, using Framer Motion for micro-animations.
- About Us Section: Detailed About Us section highlighting the company's founding, mission, story, achievements, and leadership, with 'What Sets Us Apart' cards.
- Product Showcase with Animations: Interactive product cards for Single Layer PCB, Double Layer PCB, and Multilayer PCB (Coming Soon), enhanced with Framer Motion animations and clear specifications.
- Technical Capabilities Table: Animated grid showcasing technical specifications like Laminates, PCB Types, Board Sizes, etc., using Framer Motion for smooth transitions and clear presentation.
- Contact Section with Form: Contact information (phone, email, address) and a React contact form that submits to the Express backend.
- Backend Contact API: Express API endpoint (/api/contact) that receives form data and sends email using Nodemailer, with proper error handling and CORS setup.

## Style Guidelines:

- Primary color: Strong blue (#2962FF) for reliability and innovation, as the leading tone for APSELON. Avoid teal. 
- Background color: Very light blue (#EBF0FF), for an airy and neutral backdrop.
- Accent color: Energetic orange (#FF9129) for call-to-action buttons and highlighting key elements.
- Font: 'Inter' (sans-serif) for a clean, modern, and readable experience across all text elements (headlines, body, labels, etc.)
- Framer Motion micro-interactions on hero elements, product cards, and technical tables to enhance user engagement and provide a smooth, professional feel.
- Clean and modern layout inspired by the reference image, focusing on readability and visual hierarchy.
